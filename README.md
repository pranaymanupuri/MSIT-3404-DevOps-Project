# MSIT 3404 DevOps Project

## Project Overview
This project demonstrates DevOps principles including Docker containerization and Kubernetes orchestration.

## Technologies Used
- Docker & Docker Hub
- Kubernetes (Minikube)
- Flask (Backend)
- Nginx (Frontend)

## Project Structure
- `frontend/` - Nginx web application
- `backend/` - Flask API application
- `screenshots/` - Project documentation screenshots
